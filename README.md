Myanmar Studies & DDC Reference Assistant (Readme)
📌 Project Overview
Myanmar Studies & DDC Reference Assistant သည် စာကြည့်တိုက်ပညာ၊ မြန်မာစာပေ၊ ရာဇဝင်နှင့် DDC (Dewey Decimal Classification) အညွှန်းကိန်းများကို အလွယ်တကူ ရှာဖွေနိုင်ရန် ဖန်တီးထားသော Digital Assistant တစ်ခု ဖြစ်သည်။ ဤစနစ်သည် Library 4.0 သို့ ဦးတည်သော ခေတ်မီစာကြည့်တိုက် ဝန်ဆောင်မှုတစ်ခုဖြစ်ပြီး၊ စာကြည့်တိုက်ဝန်ထမ်းများ၊ ကျောင်းသားများနှင့် သုတေသီများအတွက် အဓိကရည်ရွယ်ပါသည်။
✨ Key Features
 * Smart Search: DDC အခြေခံများ၊ စာအုပ်စာရင်းပြုစုနည်းနှင့် မြန်မာမှုဆိုင်ရာ အချက်အလက်များကို ခေါင်းစဉ် (သို့မဟုတ်) နံပါတ်ဖြင့် အမြန်ရှာဖွေနိုင်ခြင်း။
 * Ready Reference Support: အသုံးပြုသူများ သိလိုသည့် မေးခွန်းများကို တိုက်ရိုက်မေးမြန်းနိုင်ခြင်း။
 * Remote Accessibility: စာကြည့်တိုက်သို့ လူကိုယ်တိုင် လာရောက်ရန် မလိုဘဲ Work from Home အခြေအနေမျိုးတွင်ပါ အသုံးပြုနိုင်ခြင်း။
 * Integrated Inquiry System: App တွင် အဖြေမရှိသေးသော မေးခွန်းများအတွက် Google Form မှတစ်ဆင့် မေးမြန်းနိုင်ပြီး Email ဝန်ဆောင်မှုဖြင့် ပြန်လည်ဖြေကြားပေးခြင်း။
 * Admin Data Analysis: မေးခွန်းများကို စာရင်းဇယားဖြင့် ခွဲခြမ်းစိတ်ဖြာနိုင်ပြီး ဝန်ဆောင်မှုပေးထားသော အခြေအနေ (Pending/Completed) ကို Dashboard တွင် ကြည့်ရှုနိုင်ခြင်း။
🛠 Technology Stack
ဤ Project ကို အောက်ပါ နည်းပညာများကို အခြေခံ၍ တည်ဆောက်ထားပါသည် -
 * Frontend: HTML, CSS, JavaScript
 * Backend & Data Management: Excel (VBA) နှင့် Google Sheets (Database အဖြစ် အသုံးပြုနိုင်သည်)
 * Integration: Google Forms for Inquiry management
📖 How to Use
၁။ ရှာဖွေခြင်း: Search Box တွင် မိမိသိလိုသော ခေါင်းစဉ် (ဥပမာ - မြန်မာရာဇဝင်၊ ၉၅၉.၁) ကို ရိုက်ထည့်၍ ရှာဖွေပါ။
၂။ မေးမြန်းခြင်း: ရှာဖွေမှုတွင် မတွေ့ရှိပါက "မေးခွန်းရှိပါသလား?" ကဏ္ဍမှတစ်ဆင့် Google Form Link ကို နှိပ်၍ မေးခွန်းပေးပို့ပါ။
၃။ စီမံခန့်ခွဲခြင်း (Admin): Admin Panel သို့ဝင်ရောက်၍ မေးခွန်းများကို Data Analysis ပြုလုပ်ပြီး ထိရောက်စွာ ပြန်လည်ဖြေကြားပေးနိုင်ပါသည်။
